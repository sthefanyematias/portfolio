Sobre mim

Perfil autodidata com desenvolvimento técnico orientado à prática. Construo conhecimento por implementação, análise de fundamentos e evolução técnica contínua.

Atuei em projetos que cobrem desde o frontend com manipulação de DOM e Web Storage até sistemas full-stack em produção com autenticação JWT, controle de acesso por perfis (RBAC), APIs REST e containerização com Docker.

Direciono o desenvolvimento para arquitetura de soluções e entrega de valor com clareza e propósito. Valorizo processos bem estruturados, documentação técnica e ambientes colaborativos.

Stack: Java · Spring Boot · Node.js · Express.js · Angular · JavaScript · TypeScript · SQL · MongoDB · Docker · Git
